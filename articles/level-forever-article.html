Level Forever

Collect all the hearts scattered arround the forest.

About

Level Forever is a 2D platformer game that I developed for a commissioned project in under a week. The game features a heart collection mechanic, a UI that tracks the number of hearts collected, and a checkpoint system that resets the player to the last safe spot after falling into a pit or onto spikes.

UI

To manage the game UI, I created a Node called 'Container' which tracks the hearts collected. It expands as the player progresses until it fills its outline—essentially a progress bar in the shape of a heart.

Even though it looks cool, the system had a small problem: the Container Node needed to know exactly how many hearts existed beforehand to work properly. This created a coupling issue that I wanted to resolve.

Solutions

To solve this, I analyzed two simple alternatives:

Either the UI would store all the hearts as its children to maintain an active count, OR I could decouple the system by making the container simply react to collection events. In the end, I opted for an observer approach.

Implementation

I chose a hybrid approach to ensure both efficiency and clean architecture.

In my heart container class, I implemented a setup function that retrieves an array of all "Heart" type nodes to define the maximum count. Then, the container connects to a custom signal called 'heart_picked' via a Signal Bus. When the player touches a heart, the pickup class emits this signal, which the container receives to update its value automatically.

Code

Implementation in GDScript.

heart_container.gd 
 
func _setup_initial_state() -> void:
    total_hearts = get_tree().get_nodes_in_group("Heart").size()

func _connect_signals() -> void:
    SignalBus.heart_picked.connect(_increase_value)
    SignalBus.fade_in.connect(_fade_in)
    SignalBus.fade_out.connect(_fade_out)

func _increase_value() -> void:
	var new_size: float = fill.scale.x + 0.2
	hearts_collected += 1
	
	_apply_fill_update(new_size)
	
	if hearts_collected >= total_hearts:
		_handle_all_hearts_collected()

func _pick_up() -> void:
    pick_up.play()
    pick_up.reparent(get_tree().root)
    var particles : CPUParticles2D = heart_particles.instantiate()
    particles.spawn_position = global_position
    get_tree().root.add_child(particles)

    animation.play("pick_up")
    tween = get_tree().create_tween()
    tween.tween_property(sprite, "self_modulate", Color(1.0, 1.0, 1.0, 0.0), 0.6)
    SignalBus.heart_picked.emit()
    tween.tween_callback(self.queue_free)
